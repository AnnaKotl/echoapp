ADMIN PANEL for Echocode.app

  • ▹ Requests from clients:
    Click “View requests” to load all client requests from the database.
  • Use “Hide requests” to collapse the list and clear the panel view.
  • Each request has a Delete button. Confirming deletion will remove the request from the admin panel and move it to the archive in the database.
  • ▹ Configuration:
    In section "Our Products" on Home page admins can upload product icons.
  • All uploaded icons are stored in Cloudinary under the folder products-icons.
  • After upload, product icons automatically appear in the “Our Products” section of the website without additional actions.
