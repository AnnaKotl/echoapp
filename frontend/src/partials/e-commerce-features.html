Main Features of the Project

  • Category-based catalog:

    Filtered by gender, age, brand, size, and price

  • Favorites:

    Allows users to save items and return to them later

  • Cart & Checkout:

    Smooth and secure payment flow via Stripe and PayPal

  • Personal User Account:

    Track orders, update info, manage preferences

  • Order History:

    View previous purchases and order statuses

  • Real-time Shipping:

    FedEx (US) and DPD (EU)
